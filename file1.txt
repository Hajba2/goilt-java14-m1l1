git init -> створення(ініціалізація) нового репо
git status -> для перевірки наявних змін
git add filename.txt -> для додавання файлу в stagging area
git add . -> додає всі файли в stagging area
git commit -m "commit message" -> закомітити зміни в репо 
git log -> показує історію комітів
git checkout <hash> -> перейти на певний коміт
git giff <hash1> <hash2> filename.txt -> переглянути зміни між версіями файлу
